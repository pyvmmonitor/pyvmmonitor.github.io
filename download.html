PyVmMonitor is still in early beta and is only available for funders which participated in its crowdfunding campaign: https://sw-brainwy.rhcloud.com/support/pydev-2014/

Now, if you're a funder, you should already have the download link, so, download it and read along to install PyVmMonitor and its requisites:

Instaling on Windows: after downloading PyVmMonitor, double-click the installer and follow the steps!

Instaling on Linux: after downloading the .tar.gz, and moving it to the proper folder, extract it to where you'd like:

tar xvf pyvmmonitor.tar.gz

Then, create a link to start it using 'pyvmmonitor-ui' from anywhere with:

sudo ln -s `pwd`/pyvmmonitor/pyvmmonitor-ui /usr/bin/pyvmmonitor-ui

(or alternatively, just execute pyvmmonitor/pyvmmonitor-ui)

Requisites

In order to work properly at its best, PyVmMonitor also requires the current yappi development version: https://code.google.com/p/yappi/ (for the profile being thread-aware).

So, after installing PyVmMonitor, you should also install yappi and add it to your PYTHONPATH.

If you're on Windows using Python 2.7, the zip files below may be used (extract its contents so that yappi.py and _yappi.pyd are your PYTHONPATH -- usually in the python site-packages directory).

  • yappi-lib.win-amd64-2.7.zip
  • yappi-lib.win32-2.7.zip
If you have a different Python version or OS, please download/install yappi from the latest development sources (requires mercurial for 'hg'):

hg clone https://bitbucket.org/sumerc/yappi

python setup.py install

Note that it'll build a binary extension, so, make sure you have a compiler properly configured for your Python version.
