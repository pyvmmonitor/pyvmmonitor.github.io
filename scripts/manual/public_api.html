For added flexibility, it's possible to use PyVMMonitor programatically so that a profile session can be started/stopped and PyVMMonitor shows the results of the profile session afterwards.

To connect an application to PyVMMonitor (so that it's possible to start/stop profiling), use the code below (changing "PYVMMONITOR_INSTALL_DIR" to the appropriate location). 
          import sys
sys.path.append(r'PYVMMONITOR_INSTALL_DIR\public_api')
from pyvmmonitor import server
server.connect_to_pyvmmonitor()
